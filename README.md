## Introduction

Cloud computing enables applications to be hosted and accessed through the internet from anywhere in the world. In this project, a simple cloud-based Temperature Converter web application is developed using HTML, CSS, and JavaScript and deployed on the Netlify cloud platform.

The application allows users to convert temperature values between Celsius and Fahrenheit instantly using a simple and user-friendly interface.

This project demonstrates frontend web development, cloud deployment, and source code management using GitHub.

## Working of the Project

The user enters a temperature value in the input box and selects the conversion type.

The application performs:

### Celsius to Fahrenheit

Formula used:

Fahrenheit = (Celsius × 9/5) + 32

### Fahrenheit to Celsius

Formula used:

Celsius = (Fahrenheit − 32) × 5/9

The converted result is then displayed instantly on the webpage.
